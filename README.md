# TNT Tacky Light Tours - Implementation Files

Implementation files for TNT website tacky light tours promotion.

## Files

### Website Components
- **popup-example.html** - Holiday promotional popup (add to main site)
- **landing-page.html** - Tacky lights tour landing page
- **interactive-map.html** - Interactive route map

### Lead Magnets
- **TNT-Drivers-Insider-Guide.html** - Driver tips guide
- **TNT-Tacky-Lights-Checklist.html** - Tour checklist

### Data & Automation
- **tacky-lights-database.json** - Location/route data
- **n8n-workflow-tacky-lights-automation.json** - Booking automation workflow

### Assets
- **images/** - All images and graphics

## Quick Implementation

### 1. Popup (Priority)
Copy code from `popup-example.html` between START/END markers and paste before `</body>` tag on your website. Update `BOOKING_URL` variable.

### 2. Landing Page
Upload `landing-page.html` to your site (e.g., `/tacky-lights`)

### 3. Lead Magnets
Convert HTML files to PDF for download offers

### 4. Interactive Map
Upload `interactive-map.html` with `tacky-lights-database.json`

### 5. Automation
Import `n8n-workflow-tacky-lights-automation.json` into n8n
